**Readrave** provides you full understanding of how to create a website with Next.js and MDX. You can create a website with a blog, documentation, or any other content you can think of.

## Features

- Power of CLI for auto-generation of navigation, pages and components
- Full power documentation in minutes with [MDX 3](https://mdxjs.com/blog/v3) and [Next.js 14 App Directory](https://nextjs.org/docs/app)
- Advance syntax highlighting solution: Performant and reliable build-time syntax highlighting using [rehype-pretty-code](https://rehype-pretty.pages.dev) powered by [shiki](https://github.com/shikijs/shiki)
- Organize pages intuitively, with file-system routing from Next.js
- A11y as a top priority with system options, Github based markdown and much more!
